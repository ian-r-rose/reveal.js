## JupyterLab: building blocks for interactive computing * Work with components in a flexible and integrated manner * Modern JavaScript development - TypeScript - npm/yarn - WebPack - React - PhosphorJS * Clean model/view separation * Well separated public/private APIs * Fully extensible by third parties * High performance * Informed by UI/UX design and user-testing

JupyterLab: an integrated experience

## JupyterLab today * ~3.5 years of development, ~170 contributors * ~13,500 commits, ~3,700 Releases, ~60 core components * Full support for Jupyter Notebooks (the .ipynb format has not changed) * Notebooks, terminals, text editor, file browser, code console * Extension architecture enables anyone to add capabilities to JupyterLab * Integration between built-in components and extensions through public APIs * Rich handling of different data types * Ready for use! JupyterLab is now out of Beta. * `conda install -c conda-forge jupyterlab`
